# Discord-Uni
**The best discord music bot with music, leveling, games, and more.**

## Features 
- Clear Audio
- Custom Playlists
- Sharable Playlists
- Unlimited Lyrics
- Queue Support
- Youtube Search
- Open Source
- Leveling
- Monster Upgrades
- Coin System

## Setting Up (Local)

1. npm install
2. npm update
3. Edit .env
4. node .

## Setting Up (REPL.it)

1. Create New REPL (Clone Github)
2. Edit .env
3. Start
4. Set up UptimeRobot
